## Getting Started

### Prerequisites

- Node.js (v16 atau lebih baru)
- npm atau yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open browser di `http://localhost:3010`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
